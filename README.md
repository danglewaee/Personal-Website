# Dang Le Personal Website

Static portfolio site, ready for deployment.

## Files
- `index.html`
- `about.html`
- `work.html`
- `experience.html`
- `styles.css`
- `script.js`
- `vercel.json`
- `robots.txt`
- `sitemap.xml`

## Quick local preview
Open `index.html` in a browser, then use the top navigation to move across the dedicated pages.

## Deploy to Vercel
1. Push this folder to a GitHub repository.
2. Go to Vercel and import the repository.
3. Framework preset: `Other` / static site.
4. Deploy.

Vercel will automatically serve `index.html` at the root and the other `.html` files as standalone pages.

## Custom domain
After deployment, add your custom domain inside Vercel project settings.
Then replace `https://your-domain.com` in:
- `robots.txt`
- `sitemap.xml`

## Suggested domain
- `dangle.dev`
- `dang-le.dev`
